# Amazon UI Clone

A front-end clone of the Amazon homepage built using HTML and CSS.  
This project was created as part of my learning journey in web development to practice layout design, styling, and responsive UI development.

## 🚀 Features

- Amazon-style navigation bar
- Product grid layout
- Hero banner section
- Footer similar to Amazon
- Clean and structured UI

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 What I Learned

Through this project, I practiced:

- Structuring webpages using HTML
- Styling layouts using CSS
- Creating responsive sections
- Understanding how large e-commerce websites structure their UI
